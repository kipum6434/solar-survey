# Inline Edit Test Results

- การ์ดข้อมูลเทคนิคแสดงเสมอ (แม้ยังไม่มีข้อมูล แสดง "-")
- มีปุ่ม "แก้ไข" ที่มุมขวาบนของการ์ด (element 13)
- ช่องข้อมูลทั้งหมดแสดง: ขนาดระบบ, จำนวนแผง, รุ่นแผงโซล่า, อินเวอร์เตอร์, แบตเตอรี่, ทิศทางหลังคา, ค่าไฟ, ประเภทหลังคา, พื้นที่หลังคา, ระบบไฟ, ขนาดมิเตอร์, ราคาประเมิน, ราคาเสนอ
- ข้อความ "คลิกที่ช่องใดก็ได้เพื่อแก้ไขข้อมูลเทคนิค" แสดงด้านล่าง
- ต้องทดสอบคลิกเพื่อเปิด inline edit mode
