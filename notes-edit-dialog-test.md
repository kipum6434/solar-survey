# Edit Dialog Test Results

Dialog แสดงช่องข้อมูลเทคนิคครบถ้วน:
- ข้อมูลลูกค้า (แก้ไขที่หน้าลูกค้า) - แสดงเป็น read-only section ด้านบน
- สถานะและกำหนดการ: สถานะ, วันที่สำรวจ, เวลา, มอบหมายให้
- ข้อมูลทางเทคนิค (กรอกก่อนส่งให้ทีมช่าง):
  - ขนาดระบบ (kW)
  - จำนวนแผง
  - รุ่นแผงโซล่า
  - รุ่นอินเวอร์เตอร์
  - รุ่นแบตเตอรี่ (ถ้ามี)
  - ทิศทางหลังคา
  - ราคาประเมิน (บาท)
  - ราคาเสนอ (บาท)
  - หมายเหตุสำหรับช่างติดตั้ง
- หมายเหตุงานสำรวจ

ทุก field มี placeholder ภาษาไทยเป็นตัวอย่าง
